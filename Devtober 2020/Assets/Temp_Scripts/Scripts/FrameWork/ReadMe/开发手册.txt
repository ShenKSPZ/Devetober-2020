事件中心：
	输入事件：
		InputAxis(string AxisName, float AxisValue) 轴被输入
		InputButton(string ButtonName) 键被按住
		InputButtonDown(string ButtonName) 键被按下
		InputButtonUp(string ButtonName) 键被松开
	场景切换事件： 
		ProgressUpdate(float progress) 场景加载进度更新

		LeftClick(ClickInfo ClickInfo)寻路终点确定
		RightClick()

InputManager的可调参数和注意事项：
	监听InputAxis事件将收到一个object数组包含(string AxisName, float AxisValue)

UIMgr UI管理器
	分为Top Middle Bottom System四个层次
	每层同时只能有一个Panel存在
	Bottom： 游戏的探索UI 游戏的战斗UI
	Middle：互动物品菜单 对话框 调查框 扒窃框 
	Top：Esc菜单 设置菜单
	System：场景加载 系统提示
	设定上每个层的作用
	Bottom：显示持续存在的UI，譬如血条
	Middle：显示玩家通过和地图上的实物互动打开的菜单以及该菜单的子菜单
	Top：用来显示玩家按下某个按键打开的菜单以及该菜单的子菜单
	System：用来显示Loading界面 以及最高层级的系统提示的界面
